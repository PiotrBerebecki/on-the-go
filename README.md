## Getting started

```sh
# Clone the repo
$ git clone 'REPO URL'
$ cd 'REPO NAME'
$ npm install

# Start the app
$ npm start
```

## Tech stack:

- Code coverage: Istanbul and CodeCov.io
- Continuous integration: Travis CI
- Automatic deployment: Heroku
- Server: http
- Test runner: tape
- Server testing: shot
