## Getting started

```sh
# Clone the repo
$ git clone 'REPO URL'
$ cd 'REPO NAME'
$ npm install

# Start the app
$ npm start
```

## Tech stack:

- Server: http
- Test runner: tape
- Server testing: shot
- Code coverage: Istanbul and CodeCov.io
- Continuous integration: Travis CI
- Deployment: Heroku
